Política de Privacidad

1. Recogida de datos

Recogemos datos personales a través de formularios de contacto.

2. Uso de datos

Los datos se utilizan únicamente para responder consultas.

3. Protección de datos

Cumplimos con el RGPD y garantizamos la seguridad de los datos.

4. Derechos del usuario

Puedes acceder, rectificar o eliminar tus datos en cualquier momento.
